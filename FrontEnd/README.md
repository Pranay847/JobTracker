# JobTracker Frontend

React/Vite frontend for JobTracker, a full-stack job application tracker.

## What It Provides

- Authentication screens for registration and login.
- Client-side routing with React Router.
- Job application views for tracking and updating application status.
- Fast local development through Vite.

## Local Development

```powershell
npm install
npm run dev
```

Use the backend in `../BackEnd` for API and authentication routes.
